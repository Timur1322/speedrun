# 🧬 Starcraft 2 Race Predictor (Django + TensorFlow)

## 🧠 Архитектура ML-пайплайна

### Обработка сырых данных (Data Preprocessing)
Система принимает `.npz` файлы, содержащие массивы `valid_x` (признаки) и `valid_y` (зашифрованные метки).
1. **Загрузка:** Распаковка через `numpy.load`.
2. **Предикт:** Признаки передаются в модель для получения предсказаний.
3. **Дешифровка:** Истинные классы извлекаются из хешированных меток.

### Архитектура нейронной сети
Модель построена на стеке **TensorFlow/Keras**:
* **Входной слой:** Принимает векторы признаков из игровых логов.
* **Скрытые слои:** Полносвязные слои (**Dense**) с активацией **ReLU** и слоями **Dropout** для регуляризации.
* **Выходной слой:** **Softmax** (50 нейронов), выдающий вероятностное распределение по классам.

## 🔐 Алгоритм расшифровки рас (Decoding Labels)

Метки хранятся в формате: `[MD5 Хеш (32 символа)][Имя]`.
Хеш генерируется как: $MD5(\text{race\_id} + \text{name})$.

Для восстановления `race_id` (от 0 до 49) используется метод перебора:
1. Извлекается хеш и имя из строки.
2. Генерируется тестовый хеш для каждого возможного ID.
3. При совпадении ID сохраняется как истинная метка.

**Пример логики:**
```python
import hashlib as h

def get_races_from_labels(labels_data):
    races = []
    for y in labels_data:
        hi, name = y[:32], y[32:]
        for race in range(50):
            if h.md5((str(race) + name).encode()).hexdigest() == hi:
                races.append(race)
                break
    return races
```
